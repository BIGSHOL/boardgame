## 1. 디자인 철학

### 1.1 핵심 가치

| 가치 | 설명 | 구현 방법 |
|------|------|----------|
| **명확성** | 복잡한 게임 상태를 직관적으로 파악 | 정보 계층 구조, 명확한 아이콘, 색상 코딩 |
| **역사적 품위** | 조선 초기 한양의 분위기 반영 | 전통 색상, 한자/한글 혼용, 절제된 장식 |
| **기능성** | Board Game Arena 스타일의 효율적 UI | 깔끔한 레이아웃, 빠른 액션 접근, 정보 밀도 |

### 1.2 참고 서비스 (무드보드)

| 서비스 | 참고할 점 | 참고하지 않을 점 |
|--------|----------|-----------------|
| Board Game Arena | 깔끔한 2D UI, 정보 밀도, 효율적 레이아웃 | 너무 단순한 그래픽 |
| Through the Ages (BGA) | 자원 표시, 카드 UI, 턴 표시 | - |
| 창덕궁 공식 사이트 | 전통 색상, 한옥 이미지 | 현대적 레이아웃 |

## 5. 게임 UI 컴포넌트

### 5.1 게임 보드

**메인 보드 (한양 지도)**
```
┌─────────────────────────────────────┐
│  [N - 산 영역 (배산)]               │
│  ════════════════════════════       │
│                                     │
│     ┌───┬───┬───┬───┬───┐         │
│     │   │   │   │   │   │         │
│     ├───┼───┼───┼───┼───┤         │
│     │   │ 🏛 │   │   │   │         │
│     ├───┼───┼───┼───┼───┤         │
│     │   │   │   │ 🏠 │   │         │
│     ├───┼───┼───┼───┼───┤         │
│     │   │   │   │   │   │         │
│     └───┴───┴───┴───┴───┘         │
│                                     │
│  ════════════════════════════       │
│  [S - 물 영역 (임수)]               │
└─────────────────────────────────────┘
```

- 배경: Surface 컬러 (`#f8f5f0`)
- 그리드 선: Stone Gray 50% 투명도
- 산 영역: 연한 녹색 그라데이션 (`#e8f5e9`)
- 물 영역: 연한 청색 그라데이션 (`#e3f2fd`)
- 건물 타일: 해당 플레이어 컬러 보더

**개인 보드 (건축가의 책상)**
```
┌─────────────────────────────────────────────┐
│  [채석장]  [벌목장]  [기와가마]              │
│   ⛏️ +2     🪵 +2      🧱 +1                │
│   ○ ○       ○ ●       ○ ○                  │
├─────────────────────────────────────────────┤
│  자원:  🪵 2   🪨 3   🧱 1   🖌️ 2           │
│  견습공: ●●●○○  관료: ●●○○                  │
└─────────────────────────────────────────────┘
```

- 배경: 한지 텍스처 느낌의 Surface
- 슬롯: 점선 원형 (사용 가능), 실선 원형 (사용 중)
- 자원 표시: 아이콘 + 숫자, 가로 정렬

### 5.2 버튼 (Button)

| 상태 | Primary | Secondary | Ghost |
|------|---------|-----------|-------|
| 기본 | Navy 채움 + 흰 텍스트 | 테두리 + Navy 텍스트 | Navy 텍스트만 |
| 호버 | 밝은 Navy | Navy 배경 + 흰 텍스트 | 밑줄 |
| 포커스 | Gold 포커스 링 | Gold 포커스 링 | Gold 포커스 링 |
| 비활성 | 50% 투명도 | 50% 투명도 | 50% 투명도 |

**크기:**
- Large: 높이 48px, 패딩 24px (주요 액션)
- Medium: 높이 40px, 패딩 16px (기본)
- Small: 높이 32px, 패딩 12px (보조 액션)

**게임 액션 버튼:**
```
┌─────────────────────────────────────────────┐
│  [드래프팅]  [자원획득]  [건설]  [등록]     │
│                                              │
│  버튼 스타일: Secondary (기본)               │
│  선택 시: Primary로 전환                     │
│  비활성: 조건 미충족 시                      │
└─────────────────────────────────────────────┘
```

### 5.3 카드 (청사진)

```
┌─────────────────────┐
│  [건물 이미지]      │  ← 상단 60%
│       🏛️            │
│    경복궁           │
├─────────────────────┤
│  비용: 🪵5 🪨5 🧱3   │  ← 중단 20%
│  점수: 15           │
├─────────────────────┤
│  효과: 카드 2장 획득 │  ← 하단 20%
└─────────────────────┘

크기: 120px x 180px (2:3 비율)
모서리: 8px 라운드
그림자: 0 4px 8px rgba(0,0,0,0.15)
```

### 5.4 자원 표시

| 자원 | 아이콘 | 컬러 | 표시 형식 |
|------|--------|------|-----------|
| 목재 | 🪵 (통나무) | `#8b6b4d` | 아이콘 + 숫자 |
| 석재 | 🪨 (돌) | `#6b7280` | 아이콘 + 숫자 |
| 기와 | 🧱 (기와) | `#4a5568` | 아이콘 + 숫자 |
| 먹 | 🖌️ (붓) | `#1a1a1a` | 아이콘 + 숫자 |
| 점수 | ⭐ (별) | `#fdb813` | 아이콘 + 숫자 |

**증감 표시:**
- 증가: `+2` (Success 컬러)
- 감소: `-3` (Error 컬러)
- 부족: 빨간색 깜빡임

### 5.5 워커 (미플)

```
견습공 (Apprentice)        관료 (Official)
     ▲                         ▲
    /█\                       /█\ 🎩
     █                         █
    / \                       / \

크기: 24px x 32px
색상: 플레이어 컬러
관료: 갓(笠) 표시로 구분
```

### 5.6 턴/라운드 표시

```
┌─────────────────────────────────────────┐
│  라운드 3/8  │  현재 턴: 플레이어2 🔵   │
│  [◀ 이전]                   [다음 ▶]   │
└─────────────────────────────────────────┘

- 현재 플레이어: 강조 테두리 + 펄스 애니메이션
- 내 턴: Gold 배경 하이라이트
```

## 6. 반응형 레이아웃

### 6.1 브레이크포인트

| 이름 | 너비 | 기기 |
|------|------|------|
| Mobile | < 640px | 스마트폰 |
| Tablet | 640px - 1024px | 태블릿, 작은 노트북 |
| Desktop | > 1024px | 데스크톱, 대형 모니터 |

### 6.2 레이아웃 변화

**Desktop (기본)**
```
┌─────────────────────────────────────────────┐
│ [헤더: 턴 정보, 설정]                        │
├─────────┬─────────────────────┬─────────────┤
│ P1 정보 │                     │ P2 정보     │
│         │    메인 보드        │             │
│ P3 정보 │                     │ P4 정보     │
├─────────┴─────────────────────┴─────────────┤
│ [내 개인 보드]                               │
├─────────────────────────────────────────────┤
│ [액션 버튼]                                  │
└─────────────────────────────────────────────┘
```

**Tablet**
- 다른 플레이어 정보: 상단 가로 배열로 축소
- 메인 보드: 중앙 확대
- 개인 보드: 하단 스크롤 가능

**Mobile**
- 탭 기반 네비게이션 (메인 보드 / 내 보드 / 플레이어들)
- 하단 고정 액션 버튼
- 스와이프로 화면 전환

## 7. 애니메이션 & 전환

### 7.1 기본 전환

| 요소 | 전환 | 지속 시간 | 이징 |
|------|------|----------|------|
| 버튼 호버 | 배경색 | 150ms | ease-out |
| 카드 선택 | 스케일 + 그림자 | 200ms | ease-out |
| 모달 열기 | 페이드 + 스케일 | 250ms | ease-out |
| 모달 닫기 | 페이드 | 200ms | ease-in |

### 7.2 게임 애니메이션

| 액션 | 애니메이션 | 지속 시간 |
|------|-----------|----------|
| 타일 배치 | 위에서 드롭 + 바운스 | 400ms |
| 워커 이동 | 경로 따라 슬라이드 | 500ms |
| 자원 획득 | 플로팅 + 페이드 인 | 300ms |
| 점수 증가 | 숫자 롤업 + 펄스 | 600ms |
| 보너스 획득 | Gold 글로우 + 파티클 | 800ms |

### 7.3 알림 토스트

```
┌─────────────────────────────────────┐
│ ⭐ 배산임수 보너스! +3점            │
└─────────────────────────────────────┘

- 위치: 상단 중앙
- 표시: 슬라이드 다운
- 유지: 3초
- 사라짐: 페이드 아웃
```
