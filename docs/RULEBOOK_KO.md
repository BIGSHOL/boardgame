# 한양: 도읍의 설계자들 - 게임 룰북

> 1394년, 조선 태조 이성계는 새로운 수도 한양의 건설을 명했습니다.
> 당신은 한양 건설의 총책임자로서, 궁궐, 관아, 시전, 민가를 건설하고
> 풍수지리를 활용하여 가장 번영하는 도시를 만들어야 합니다.

## 1. 게임 개요

**플레이어 수:** 2-4명
**플레이 시간:** 약 45-60분
**라운드 수:** 4라운드

### 게임 목표

조선 초기 한양의 건설에 참여하여 **가장 높은 점수**를 획득하세요.

점수는 다음을 통해 획득합니다:
- 건물 타일 배치
- 풍수지리(배산임수) 보너스
- 인접 시너지 보너스
- 청사진 카드 완성
- 배치된 일꾼

## 2. 게임 구성물

### 게임 보드

**5x5 격자판**으로 구성된 한양 지도입니다.

| 지형 | 위치 | 설명 |
|------|------|------|
| 산(山) | 보드 모서리 4곳 | 건물 배치 불가 |
| 물(水) | 보드 중앙 (청계천) | 건물 배치 가능, 풍수 보너스 제공 |
| 평지 | 나머지 칸 | 건물 배치 가능 |

```
[ 산 ][  ][  ][  ][ 산 ]
[    ][  ][  ][  ][    ]
[    ][  ][ 물 ][  ][    ]
[    ][  ][  ][  ][    ]
[ 산 ][  ][  ][  ][ 산 ]
```

### 자원 토큰

| 자원 | 한글명 | 최대 보유량 |
|------|--------|------------|
| 🪵 Wood | 목재 | 10개 |
| 🪨 Stone | 석재 | 10개 |
| 🧱 Tile | 기와 | 6개 |
| 🖋️ Ink | 먹 | 4개 |

### 일꾼

| 일꾼 | 시작 수 | 자원 생산량 | 특징 |
|------|---------|------------|------|
| 견습생 (Apprentice) | 3명 | 1개 | 타일당 2칸 배치 가능 |
| 관리 (Official) | 2명 | 2개 | 타일당 1칸 배치 가능 |

### 건물 타일

총 **36개**의 건물 타일이 있습니다:

| 카테고리 | 개수 | 자원 생산 | 특징 |
|----------|------|----------|------|
| 궁궐 (Palace) | 4개 | - | 고득점, 고비용 |
| 관아 (Government) | 6개 | 먹 | 중득점, 궁궐 시너지 |
| 사찰 (Religious) | 6개 | 기와 | 중득점, 풍수 보너스 |
| 시전 (Commercial) | 8개 | 석재 | 저득점, 저비용 |
| 민가 (Residential) | 8개 | 목재 | 저득점, 저비용 |
| 성문 (Gate) | 4개 | - | 특수 효과 |

### 청사진 카드

총 **24장**의 청사진 카드가 있으며, 게임 종료 시 조건 충족에 따라 추가 점수를 제공합니다.

## 3. 게임 준비

### 단계별 준비

1. **게임 보드 준비**
   - 5x5 보드를 중앙에 놓습니다
   - 모서리 4칸은 산 지형으로 건물 배치가 불가합니다
   - 중앙 칸은 물(청계천) 지형입니다

2. **건물 타일 준비**
   - 36장의 건물 타일을 잘 섞습니다
   - 뒷면이 보이도록 쌓아서 "건물 더미"를 만듭니다
   - 상위 3장을 공개하여 "건물 시장"을 구성합니다

3. **플레이어 준비**
   - 각 플레이어는 다음을 받습니다:
     - **초기 자원**: 목재 2개, 석재 2개
     - **일꾼**: 견습생 3명, 관리 2명
     - **색상 마커**: 자신의 색상 선택

4. **청사진 카드 배분**
   - 청사진 카드를 잘 섞습니다
   - 각 플레이어에게 **3장씩** 배분합니다
   - 배분받은 카드 중 **1장을 선택**하여 비공개로 보관합니다
   - 나머지 2장은 버립니다

5. **시작 플레이어 결정**
   - 무작위로 시작 플레이어를 정합니다
   - 턴 순서는 시계 방향으로 진행됩니다

---

## 4. 게임 진행

### 게임 흐름

게임은 **4라운드**로 진행됩니다. 각 라운드에서 모든 플레이어가 턴을 가집니다.

```
라운드 1 → 라운드 2 → 라운드 3 → 라운드 4 → 게임 종료
   ↓          ↓          ↓          ↓
 모든 플레이어  모든 플레이어  모든 플레이어  모든 플레이어
   턴 진행      턴 진행      턴 진행      턴 진행
```

### 라운드 구조

1. 시작 플레이어부터 시계 방향으로 턴 진행
2. 각 플레이어는 자신의 턴에 액션 수행
3. 모든 플레이어가 턴을 마치면 라운드 종료
4. 다음 라운드 시작

---

## 5. 턴 구조와 액션

### 턴 진행

자신의 턴에 다음 액션을 **자유롭게** 수행할 수 있습니다:

| 액션 | 설명 | 필수 여부 |
|------|------|----------|
| 건물 배치 | 건물 타일을 보드에 배치 | 선택 |
| 일꾼 배치 | 일꾼을 건물 위에 배치 | 선택 |
| 청사진 선택 | 배분받은 청사진 중 선택 | 선택 |
| 턴 종료 | 턴을 종료하고 자원 수집 | **필수** |

### 턴 종료 시

**턴 종료**를 선택하면:

1. **자원 수집**: 배치된 일꾼이 있는 건물에서 자원을 수집합니다
   - 견습생: 1개의 자원
   - 관리: 2개의 자원

2. **다음 플레이어**에게 턴이 넘어갑니다

## 6. 건물 배치

### 건물 배치 방법

1. **건물 선택**
   - 건물 시장(공개된 상위 3장)에서 원하는 건물을 선택합니다

2. **비용 지불**
   - 선택한 건물의 비용을 자원으로 지불합니다

3. **위치 선택**
   - 보드의 빈 칸(산 지형 제외)에 건물을 배치합니다

4. **점수 획득**
   - 건물의 기본 점수를 획득합니다
   - 풍수 보너스와 인접 보너스가 적용됩니다

### 배치 제한

- **산 지형**에는 건물을 배치할 수 없습니다
- 이미 **건물이 있는 칸**에는 배치할 수 없습니다
- 비용을 지불할 **자원이 부족**하면 배치할 수 없습니다

### 인접 보너스

특정 건물은 다른 종류의 건물과 인접할 때 추가 점수를 받습니다:

| 건물 | 인접 보너스 대상 | 보너스 점수 |
|------|-----------------|------------|
| 궁궐 | 관아 | +1~2점 |
| 관아 | 궁궐 | +1~2점 |
| 시전 | 시전, 민가 | +1점 |
| 민가 | 궁궐, 시전 | +1점 |

---

## 7. 일꾼 배치

### 일꾼 배치 방법

1. **일꾼 선택**
   - 보유한 견습생 또는 관리 중 하나를 선택합니다

2. **배치 위치 선택**
   - **건물이 있는 칸**에만 배치할 수 있습니다
   - 빈 칸이나 산 지형에는 배치할 수 없습니다

3. **슬롯 확인**
   - 각 건물에는 일꾼 슬롯이 있습니다:
     - **견습생 슬롯**: 2칸
     - **관리 슬롯**: 1칸
   - 이미 채워진 슬롯에는 배치할 수 없습니다

### 자원 생산

| 건물 유형 | 생산 자원 |
|----------|----------|
| 관아 (Government) | 🖋️ 먹 |
| 사찰 (Religious) | 🧱 기와 |
| 시전 (Commercial) | 🪨 석재 |
| 민가 (Residential) | 🪵 목재 |
| 궁궐 (Palace) | 자원 생산 없음 |
| 성문 (Gate) | 자원 생산 없음 |

### 생산량

- **견습생**: 1개의 자원
- **관리**: 2개의 자원

예시: 민가에 관리 1명을 배치하면 턴 종료 시 목재 2개를 획득합니다.

## 8. 청사진 카드

### 청사진 카드란?

청사진 카드는 게임 종료 시 특정 조건을 충족하면 **추가 점수**를 제공합니다.

### 청사진 선택

1. 게임 시작 시 3장의 청사진을 배분받습니다
2. 게임 중 언제든지 1장을 선택할 수 있습니다
3. 선택하지 않은 카드는 버려집니다

### 청사진 카테고리

| 카테고리 | 조건 예시 | 보너스 점수 |
|----------|----------|------------|
| 궁궐 인접 | 궁궐에 인접한 내 건물 2개 이상 | 4~10점 |
| 건물 수집 | 특정 유형 건물 3~4개 이상 | 4~7점 |
| 패턴 | 한 줄에 내 건물 4개 이상 | 4~7점 |
| 특수 | 풍수 보너스 건물 3개 이상 | 4~8점 |

### 주요 청사진 카드

**궁궐 인접 계열:**
- **궁궐의 벗** (4점): 궁궐에 인접한 내 건물 2개 이상
- **궁성 수호자** (6점): 궁궐에 인접한 내 건물 3개 이상
- **어전 담당관** (10점): 궁궐 4방향 모두에 내 건물 배치

**건물 수집 계열:**
- **상업 거물** (6점): 시전 건물 4개 이상
- **관료의 길** (5점): 관아 건물 3개 이상
- **만물상** (7점): 5가지 다른 건물 유형 보유

**패턴 계열:**
- **가로 완성** (5점): 한 가로줄에 내 건물 4개 이상
- **밀집 지구** (6점): 2x2 영역에 내 건물 4개
- **모서리 장악** (5점): 보드 모서리 4곳 중 3곳에 내 건물

**특수 계열:**
- **풍수 달인** (6점): 풍수 보너스 받는 건물 3개 이상
- **연결 제국** (8점): 모든 내 건물이 서로 인접 연결

## 9. 풍수지리 보너스

### 배산임수 (背山臨水)

조선시대 풍수지리의 핵심 원칙입니다:
- **배산(背山)**: 뒤에 산을 두다
- **임수(臨水)**: 앞에 물을 바라보다

### 풍수 보너스 조건

건물 배치 시 다음 조건을 충족하면 풍수 보너스를 받습니다:

| 조건 | 보너스 |
|------|--------|
| 북쪽(윗줄)에 산 + 물 인접 | 100% 풍수 보너스 |
| 북쪽에 산만 있음 | 50% 풍수 보너스 |
| 물에만 인접 | 50% 풍수 보너스 |

### 건물별 풍수 보너스 점수

| 건물 유형 | 풍수 보너스 (100%) |
|----------|-------------------|
| 궁궐 | 3~4점 |
| 관아 | 1~2점 |
| 사찰 | 1~3점 |
| 시전 | 0~1점 |
| 민가 | 0~2점 |
| 성문 | 1~2점 |

예시: 경복궁(풍수 보너스 4점)을 산 아래, 물 옆에 배치하면 기본 점수 8점 + 풍수 보너스 4점 = **12점**을 획득합니다.

## 10. 자원 관리

### 초기 자원

게임 시작 시 각 플레이어는:
- 🪵 목재 2개
- 🪨 석재 2개
- 🧱 기와 0개
- 🖋️ 먹 0개

### 자원 획득 방법

1. **일꾼 생산**: 턴 종료 시 배치된 일꾼이 자원 생산
2. **특수 효과**: 일부 건물의 특수 효과로 획득

### 자원 상한선

| 자원 | 최대 보유량 |
|------|------------|
| 🪵 목재 | 10개 |
| 🪨 석재 | 10개 |
| 🧱 기와 | 6개 |
| 🖋️ 먹 | 4개 |

**주의**: 상한선을 초과하는 자원은 소멸됩니다!

### 자원 사용

자원은 **건물 배치 비용**으로 사용됩니다.

예시 건물 비용:
| 건물 | 비용 |
|------|------|
| 경복궁 | 목재 3 + 석재 3 + 기와 2 + 먹 1 |
| 의정부 | 목재 2 + 석재 2 + 먹 1 |
| 시전 | 목재 1 + 석재 1 |
| 북촌 | 목재 2 |

## 11. 게임 종료 및 점수 계산

### 게임 종료 조건

다음 중 하나가 발생하면 게임이 종료됩니다:

1. **4라운드 완료**: 모든 플레이어가 4번째 라운드를 마침
2. **건물 소진**: 건물 더미가 모두 소진됨

### 최종 점수 계산

| 항목 | 설명 | 점수 |
|------|------|------|
| 기본 점수 | 건물 배치로 얻은 점수 | 누적 합산 |
| 청사진 보너스 | 청사진 조건 충족 시 | 카드별 상이 |
| 일꾼 점수 | 배치된 일꾼 1명당 | +1점 |
| 자원 패널티 | 남은 자원 3개당 | -1점 |

### 점수 계산 예시

```
플레이어 A의 최종 점수:
- 기본 점수: 25점 (건물 배치)
- 청사진 보너스: +6점 (상업 거물 완성)
- 일꾼 점수: +4점 (4명 배치)
- 자원 패널티: -2점 (6개 남음)
----------------------------
총점: 33점
```

### 승자 결정

**가장 높은 점수**를 획득한 플레이어가 승리합니다.

**동점 시 처리:**
1. 배치한 건물 수가 많은 플레이어
2. 남은 자원이 적은 플레이어
3. 그래도 동점이면 공동 승리

## 게임 팁

### 초보자를 위한 팁

1. **자원 관리**: 초반에는 민가와 시전을 지어 자원을 확보하세요
2. **일꾼 활용**: 관리는 생산량이 2배이므로 효율적으로 배치하세요
3. **풍수 보너스**: 물과 산 근처에 건물을 배치하면 추가 점수를 얻습니다
4. **청사진**: 달성 가능한 청사진을 선택하세요

### 전략 가이드

**자원 집중 전략**
- 민가/시전 위주로 건설
- 일꾼을 적극 활용하여 자원 축적
- 후반에 고가 건물(궁궐) 건설

**점수 집중 전략**
- 풍수 보너스가 높은 위치에 궁궐 배치
- 인접 보너스를 최대화하는 배치
- 청사진 조건 충족에 집중

**청사진 특화 전략**
- 달성 쉬운 청사진 선택
- 청사진 조건에 맞춰 건물 배치
- 여러 청사진 조건 동시 충족 노림
